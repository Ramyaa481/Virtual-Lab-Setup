# Virtual-Lab-Setup
Virtual Cybersecurity Lab Setup Using Kali Linux

## Objective
To create a safe and isolated environment for learning cybersecurity concepts using Kali Linux and VirtualBox.

## Tools Used
- Kali Linux
- Oracle VirtualBox
- Windows 11

## Procedure
1. Downloaded VirtualBox.
2. Downloaded Kali Linux ISO.
3. Created a virtual machine.
4. Installed Kali Linux.
5. Configured network settings.
6. Verified installation using terminal commands.

## Commands Used
whoami

uname -a

ip a

## Outcome
Successfully created a virtual cybersecurity lab using Kali Linux in VirtualBox.
